# ScriptLangs_2026.Task-1.Template

## 2026: Задание для студентов МИИГАиК и ИПМиКН - Вариант 1 (сложный) - засчитывается как 2 лабы - и Python и JS

Используя ЯП Python 3.14 для бэкенд реализации и Javascript для UI разработать минималистичный сервис генерации PDF документов с подстановкой шаблонных значений.
При открытии корневого пути / должна отображаться страница с wysiwyg редактором html (напр tinymce, monaco, Quill.js) и окно отображения PDF документа. Пользователь вводит шаблон страницы и добавляет в него маркеры для шаблонизатора {value}.
По запросу (POST /render/{id}) сервис получает набор параметров и совершает подстановку в сохраненный шаблон, после чего генерирует и возвращает итоговый PDF документ.
Для хранения документов использовать встроенную БД sqlite или локальные файлы в формате json. Подстановка шаблонизатора должно использовать t-strings (обратите внимание, доступны только в Python 3.14) и для генерации pdf использовать библиотеку xhtml2pdf (Pisa)
API endpoints:
GET / -> возвращает html с UI
GET /templates
POST /templates
GET /templates/{id}
PUT /templates/{id}
DELETE /templates/{id}
POST /render/{id} body: {field: value} -> PDF download
GET /preview/{id} body: {field: value} -> HTML preview с подстановкой
